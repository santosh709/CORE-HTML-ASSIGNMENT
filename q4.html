List of Tags.

  1. div and span: Used to communicate the function of blocks of content
  2. h1: Defines a heading level 1.
  3. Link: Used to connect a document to a related resource.
    4. blockquote: Used to indicate that content is from an outside source.
  4. p: Defines a paragraph of text.
  5. div: Defines a container for HTML elements.
  6. list: Used to group related pieces of information.
  7. table: Defines a table with rows and columns.
  8. ul: Defines an unordered list.
  9. meta: Used inside the head tag to describe metadata.
  10. img: Used to put images in HTML files. The tag has no closing tag.
